OWASP Norway Day

November 20th 2018

Call for Papers is open

Closes 5th of August 2018

Submit here: https://easychair.org/conferences/?conf=owaspno2018

The OWASP Norway Chapter is 10 years this year, and we are celebrating this with a one-day single-track conference at the University of Oslo. The talks will be selected after the Call for Papers closes, and the selected topics are DevSecOps, Blue team, Cloud Security, End User Security, Mobile Application Security, API security, Integrations, War stories, Role integration, Secure Software Development Lifecycle, Secure Code Assessments, Security Architecture and gerenal Web Application Security.

The conference will be targeted at developers, security engineers and security testers, and will be limited to 200 participants.

If you are interested in sponsoring the event, please contact: kelly.santalucia@owasp.org
